==================
Mrp Order Planning
==================

MRP Orders planning report. This report visualizes the selected orders based on the attributes of the variants.

Installation
============

To install this module, you need to:

#. Go to application and install module.

Usage
=====

To use this module, you need to:

#. Go to Manufacture orders ans select it. On print menu choice Report by variants.
